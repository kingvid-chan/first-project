ie8兼容

1、ie8下opacity无效，用filter: progid:DXImageTransform.Microsoft.Alpha(opacity=0)代替（
有个小问题就是：如果是使用  filter:alpha(opacity=0); ie浏览器的opacity属性会继承给子元素。）
2、设置ie6/7/8浏览器背景层颜色透明或者颜色渐变：
filter:progid:DXImageTransform.Microsoft.Gradient(startColorStr=#AARRGGBB,endColorStr=#AARRGGBB);
其中， 其格式为 #AARRGGBB 。 AA 、 RR 、 GG 、 BB 为十六进制正整数。取值范围为 00 – FF 。 RR 指定红色值， GG 指定绿色值， BB 指定蓝色值，参阅 #RRGGBB 颜色单位。 AA 指定透明度。 00 是完全透明。 FF 是完全不透明。超出取值范围的值将被恢复为默认值。
3、ie8下要去掉carousel-control箭头下背景的渐变，需要设置filter:none;
4、注意：ie8不支持：last-child选择器；
5、IE HACK之@media hack：
① 针对IE 8的hack：
@media \0screen {
    body { background: blue; }
② 针对 IE 6 and 7
@media screen\9 {

③ 针对 IE 6, 7 and 8
@media \0screen\,screen\9 {
    body { background: green; }} 
④ 针对IE 8, 9 and 10
@media screen\0 {
    body { background: orange; }} 
⑤ 针对IE 9 and 10
@media screen and (min-width:0\0) {
    body { background: yellow; }}
⑥ 针对 IE 10
@media screen and (-ms-high-contrast: active), (-ms-high-contrast: none) { 
   /* IE10-specific styles go here */ 
 }

二、@media(min-aspect-ratio:1440/724)兼容ie及图片自适应铺满屏幕

    var bodyw = $(window).width();
    var bodyh = $(window).height();
    if(bodyw/bodyh >= 1440/724) {  //图片宽高比例
        $('.item > img').css({'width':'100%','height':'auto'});
    }else {
        $('.item > img').css({'height':'100%','width':'auto'});
    }

四、position与margin关系

解决方法：将 margin: auto 替换成 margin-left: auto; margin-right: auto;

    <div>
        <img src="pic.jpg">
    </div>
    img {
        position: absolute;
        bottom: -85px;
        left: 0;
        right: 0;
        width：40px;
        margin: auto;  //ie中垂直居中显示，bottom: -85px无效
    }


五、css3动画hover动画与overflow溢出问题

    <div>
        <img src="pic.jpg" alt="">  
    </div>
    img {
        width: 100%;
        height: 100%;
        transition: all 1s;
        z-index: 1;
    }
    img:hover { 
        transform: scale(1.5);  //动画执行，图片放大会超过div的圆角溢出，overflow和z-index无效
    }
    div {
        width: 200px;
        height: 200px;
        border-radius: 500px;
        overflow: hidden;
        z-index:2;
        -webkit-transform-style: preserve-3d;
        -o-transform-style: preserve-3d;
        -moz-transform-style: preserve-3d;
        -ms-transform-style: preserve-3d;
        transform-style: preserve-3d;   //解决动画执行时的溢出问题
    }

    transform-style属性: 
    问题：img变大，超出div圆形范围，z-index无效；
    原因：有些浏览器默认transform-style：flat；flat值为默认值，表示所有子元素在2D平面呈现，则该元素的所有子元素都将被平展到该元素的2D平面中进行呈现。导致位于正或负Z轴位置的子元素显示在该元素的平面上，而不是它的前面或者后面，z-index无效。
    preserve-3d表示所有子元素在3D空间中呈现，元素分前后关系，z-index有效。
